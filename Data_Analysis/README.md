# Data Analysis & Visualization

## 📌 Description
A beginner Python project that performs advanced exploratory data analysis (EDA) on a sample dataset, providing statistical insights and visualizations using Pandas and Matplotlib.

## 🚀 Features
- Load and display dataset  
- Perform statistical analysis (mean, median, sum, etc.)  
- Visualize data with line, bar, and histogram graphs  
- Identify patterns and trends  

## 🛠️ Concepts Used
- Pandas for data manipulation  
- Matplotlib for data visualization  
- Basic EDA and statistical analysis  

## ▶️ How to Run
1. Download the project  
2. Open terminal/command prompt in the project folder  
3. Run the program:
   ```bash
   python analysis.py
