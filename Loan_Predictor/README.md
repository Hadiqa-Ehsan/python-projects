# Loan Eligibility Predictor

## 📌 Description
A beginner Python project that predicts whether a user is eligible for a loan using Logistic Regression, demonstrating basic binary classification techniques.

## 🚀 Features
- Predict loan eligibility (Yes/No)  
- Train and test a Logistic Regression model  
- Check model accuracy on dataset  

## 🛠️ Concepts Used
- Logistic Regression from Scikit-learn  
- Binary classification basics  
- Model evaluation  

## ▶️ How to Run
1. Download the project  
2. Open terminal/command prompt in the project folder  
3. Run the program:
   ```bash
   python loan_predictor.py
