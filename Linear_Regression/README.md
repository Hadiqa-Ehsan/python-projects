# Linear Regression Project

## 📌 Description
A beginner Python project that predicts student scores based on study hours using a simple Linear Regression model, with visualizations to understand the data and predictions.

## 🚀 Features
- Train a Linear Regression model  
- Make predictions for given study hours  
- Visualize data and regression line with graphs  

## 🛠️ Concepts Used
- Scikit-learn  
- Regression analysis  
- Matplotlib for data visualization  

## ▶️ How to Run
1. Download the project  
2. Open terminal/command prompt in the project folder  
3. Run the program:
   ```bash
   python linear_regression.py
