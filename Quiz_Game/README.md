# Quiz Game

## 📌 Description
A beginner Python project that presents multiple questions to the user and tracks their score.

## 🚀 Features
- Multiple questions with answers  
- Tracks user score  
- Handles user input  

## 🛠️ Concepts Used
- Lists  
- Loops  
- Conditional statements (if-else)  

## ▶️ How to Run
1. Download the project  
2. Open terminal/command prompt in the project folder  
3. Run the program:
   ```bash
   python quiz_game.py
