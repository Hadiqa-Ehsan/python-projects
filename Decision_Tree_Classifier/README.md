# Decision Tree Classifier

## 📌 Description
A beginner Python project that predicts whether a student will pass or fail based on study data using a Decision Tree classifier, with visualizations to understand the model's decisions.

## 🚀 Features
- Train a Decision Tree classification model  
- Make predictions for new student data  
- Visualize the decision tree for better understanding  

## 🛠️ Concepts Used
- Decision Tree Classifier from Scikit-learn  
- Classification techniques  
- Data visualization basics  

## ▶️ How to Run
1. Download the project  
2. Open terminal/command prompt in the project folder  
3. Run the program:
   ```bash
   python decision_tree.py
