# Customer Data Analysis

## 📌 Description
A beginner Python project that performs basic data analysis on a sample customer dataset using Pandas, providing insights through statistics and visualizations.

## 🚀 Features
- Load and display dataset  
- Calculate mean, median, sum, and other basic statistics  
- Visualize data with graphs  
- Identify top spenders and patterns  

## 🛠️ Concepts Used
- Pandas  
- DataFrames  
- Data analysis and visualization basics  

## ▶️ How to Run
1. Download the project  
2. Open terminal/command prompt in the project folder  
3. Run the program:
   ```bash
   python data_analysis.py
