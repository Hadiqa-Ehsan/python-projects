# Spam Classifier

## 📌 Description
A beginner Python project that classifies text messages as spam or not spam using Naive Bayes, demonstrating basic text classification techniques.

## 🚀 Features
- Classify messages as spam or not spam  
- Make predictions on new text input  
- Simple text preprocessing and vectorization  

## 🛠️ Concepts Used
- Naive Bayes classifier  
- CountVectorizer from Scikit-learn  
- Text classification basics  

## ▶️ How to Run
1. Download the project  
2. Open terminal/command prompt in the project folder  
3. Run the program:
   ```bash
   python spam_classifier.py
