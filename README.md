# Python Beginner Projects Portfolio

## 👋 Welcome
This repository contains **9 beginner Python projects** that demonstrate fundamental programming concepts, data analysis, and problem-solving skills.

## 🚀 Projects Included

1. **Personal Budget Tracker**  
   Track monthly income and expenses, calculate remaining budget, and save data to CSV.

2. **Quiz Game**  
   Multiple-choice quiz game with score tracking.

3. **Password Generator**  
   Generate random strong passwords with letters, numbers, and symbols.

4. **Customer Data Analysis**  
   Perform basic EDA on customer data using Pandas and visualize trends.

5. **Linear Regression Project**  
   Predict student scores based on study hours with regression and visualization.

6. **Spam Classifier**  
   Classify messages as spam or not spam using Naive Bayes.

7. **Decision Tree Classifier**  
   Predict pass/fail using study data and visualize the decision tree.

8. **Loan Eligibility Predictor**  
   Predict loan eligibility using Logistic Regression.

9. **Data Analysis & Visualization**  
   Advanced EDA with statistical insights and graphs (line, bar, histogram).

## 🛠️ Skills & Concepts
- Python basics: Variables, Loops, Lists, Functions  
- File Handling & CSV  
- Data Analysis & Visualization (Pandas, Matplotlib, Seaborn)  
- Machine Learning basics (Regression, Classification)
